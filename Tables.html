Tables

Tables are only suppose to show the datas. Here is a example of a simple table.

OneTwoThree
FourFiveSix
SevenEightNine

Table headings:

If you want to have a table heading for each column, use th: table headings. Notice that the table heading should be inside a table row. Example below:

Row OneRow TwoRow Three
OneTwoThree
FourFiveSix
SevenEightNine

Row Span / Col Span:

Row span attributes or col span attributes are used to combine two rows or columns. See result of below:

Child's Name: Parent's Name:
CatherineMichael McCarthy
Colleen McCarthy
MaggieSheila McGee
EdwardCatherine Howard
Jeff Howard

Caption

Caption is an element that provides a caption or title for a table. It should goes right after the table tag and is positioned at top of the table by default.
