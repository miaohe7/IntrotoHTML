Block Tags

Block tags, means it will have a new line before and after the tag. It will also take up all the given width.

hr

The hr element means "horizontal line". It is useful to break up the page. In HTML5 it took on a semantic meaning, "semantic break." It is self-closing, which means it has no closing tag.

address

The address element displays the address. The display depends on the browser that the user use, for example, if using Chrome it shows the text as Italic. And it is useful for people using screen reader to find the address section.

University of Waterloo 200 University Avenue West Waterloo, Ontario, N2L 3G1

blockquote

Blockquote element is used for quotes, it should include the cite attribute in order to show or link to the source of the quote. Also, it will automatically indent.

"We can only see a short distance ahead, but we can see plenty there that needs to be done." - Alan Turing

details

Details element is used to conjunction with the summary tag.

Summary

If you need to click to see this, it means your browser supports the details element.

Inline Tags

Inline elements can flow next to each other and don't cause a line break before and after this element.

Citing a Source - cite

The cite tag makes it much easier for people and bots to search the code for this specific information.

The Rizzoli & Isles by Tess Gerrits

Abbreviation

The abbr tag display abbreviation on the browser, but show the full name when mouse hover over the abbreviation.

The month of the year are often abbreviated: JA, FE, APR, etc.

The abbr tag shows a dotted underline by default in several browsers, it need a line of CSS. We will get to it later.

Time

time is a completely semantic tag that can be used to search for specific info.

Dinner at 7:00.

I have a date on Valentines day.

Code

code element tends to display in a different font, but also semantic meaning to convey information to as many people as possible.

I'm using code in my paragraph.

Subscrpt and Superscripts

sub and sup are a lot of times used in mathematical formulas. For example, H2O and x2y3.
